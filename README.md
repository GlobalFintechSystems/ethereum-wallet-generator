# Ethereum wallet generator

## Use with npx
```
$ npx eth-wallet-generator
```

## Install
```
npm install -g eth-wallet-generator
```

## Usage:
```
$ eth-wallet-generator
```

1. You will be prompted with password input
2. Save mnemonic and private key on a piece of paper

Output:
```javascript

```
